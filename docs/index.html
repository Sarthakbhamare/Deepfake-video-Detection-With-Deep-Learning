About this Project

This static site is hosted on GitHub Pages. The full application is a Django backend that performs video processing and model inference with PyTorch. Because GitHub Pages is static-only, the live prediction feature requires a separate backend deployment.

Run Locally
  1. Install Python 3.10+ and Git.
  2. Clone the repo and create a venv.
  3. Install dependencies from Django Application/requirements.txt.
  4. Run migrations and start the server.
git clone https://github.com/Sarthakbhamare/Deepfake-video-Detection-With-Deep-Learning.git
cd "Deepfake-video-Detection-With-Deep-Learning"
python -m venv .venv
.\.venv\Scripts\activate
pip install --upgrade pip
pip install -r "Django Application/requirements.txt"
cd "Django Application"
python manage.py migrate
python manage.py runserver

Then open http://127.0.0.1:8000 in your browser.

Backend Deployment Options

To enable online predictions from this Pages site, deploy the Django backend on a host and enable CORS:

  • Container hosts: Render, Railway, Fly.io, AWS, Azure App Service
  • Then update this page to point to your API endpoint

Note: The current Pages site is informational only; predictions require a running backend.
